NOW WE CAN START INSTALLATION FOR ULTRAMARINE 42 VIA WSL. FIRST THING UPDATE ALL AND UPGRADE VIA ROOT USER.

---------------------------------

After installation from Microsoft STORE - Run this for ULTRAMARINE 42 as ROOT user.

sudo dnf5 upgrade --refresh

----------------------------------

Upgrade Ultramarine from 41 to 42 as ROOT user.

Copy and paste in terminal and press y if asked.

sudo dnf upgrade --refresh
sudo dnf distro-sync --releasever=42

----------------------------------

Reboot Ultramarine 42 - Reboot Ultramarine 42 - Reboot Ultramarine 42

LOGIN AS NORMAL USER IN TERMINAL AND INSTALL PROGRAMS and GNOME......

------------------------------------------------------

Install some other programs for Ultramarine 42

sudo dnf5 install curl wget git bash zeal demonized chattr librepo --skip-unavailable -y

------------------------------------------------------

Fix IP problem for GWSL or X410. Cant find screen problems. Look ip adress in windows terminal / PowerShell as root : ipconfig

NOTE - Importent to look for ip. LAST LINES IN WINDOWS TERMINAL IS IP FOR WSL

NOW GO BACK TO ULTRAMARINE TERMINAL AND ENTER THIS...

sudo ls -l /etc/resolv.conf
sudo mv /etc/resolv.conf /tmp/resolv.conf.bak
sudo nano /etc/resolv.conf

nameserver 172.20.208.1 <-- put your ip here and add nameserver.

sudo chattr +i /etc/resolv.conf
sudo systemctl restart systemd-resolved
sudo cat /etc/resolv.conf
