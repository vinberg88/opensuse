Make small script for Ubuntu 26.04 LTS and UKUI desktop.

Copy this and paste in terminal: sudo nano ~/.bash_ubuntu_desktop

Then add this lines and save in terminal

export XDG_CURRENT_DESKTOP=ubuntu:GNOME
export XDG_SESSION_DESKTOP=ubuntu
export DESKTOP_SESSION=ubuntu
export GNOME_SHELL_SESSION_MODE=ubuntu

export XDG_CONFIG_DIRS=/etc/xdg/xdg-ubuntu:/etc/xdg
export XDG_DATA_DIRS=/usr/share/ubuntu:/usr/local/share:/usr/share:/var/lib/snapd/desktop
export XDG_MENU_PREFIX=gnome-

export XDG_SESSION_TYPE=x11
export XDG_SESSION_CLASS=user
export GDK_BACKEND=x11

export LIBGL_ALWAYS_SOFTWARE=1

----------------------------------------------------------------

Now we must install scrip for UKUI / KYLIN desktop and Ubuntu 26.04 LTS

Copy and paste in terminal - one by one =]


sed -i 's/\r$//' /tmp/ubuntu-UKUI.sh

bash -n /tmp/ubuntu-UKUI.sh && echo "Syntax OK"

sudo mkdir -p ~/.local/bin

sudo chown -R "$(id -un):$(id -gn)" ~/.local

install -m 755 /tmp/ubuntu-UKUI.sh ~/.local/bin/ukui-x410

ls -l ~/.local/bin/ukui-x410

grep -qxF 'export PATH="$HOME/.local/bin:$PATH"' ~/.bashrc || echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc

source ~/.bashrc

command -v ukui-x410
