TIME TO START THIS PROJECT SO LETS GO - FEDORA 44 

LOGIN AS ROOT.... FEDORA 44 - KDE 6

Update Fedora 44

sudo dnf5 upgrade --refresh

-----------------------------------

Install Nano - copy and paste in terminal - Fedora 44

sudo dnf install nano

-----------------------------------

Install some basic programs for fedora 44.

sudo dnf5 install systemd* git wget curl bash pipx net-tools dbus-x11 dbus* yum -y --allowerasing --skip-broken

-----------------------------------

LOGIN AS ROOT and setup password and User.

Add password for root: passwd

Now add new user: adduser vinberg <-- take your name =]

Add password for user: passwd vinberg

-----------------------------------

Paste in terminal: sudo nano /etc/wsl.conf

Add this to Bash 

[boot]
systemd=true

And add your user like this

[user]
default=vinberg

-----------------------------------

Add user to sudoers file: sudo nano /etc/sudoers

LIKE THIS....

root    ALL=(ALL:ALL) ALL
vinberg ALL=(ALL:ALL) ALL

-----------------------------------

Install fastmirrors for fedora 44

Run this first

sudo dnf check-update
sudo dnf install dnf-utils

Then do this..

sudo nano /etc/dnf/dnf.conf

Add the following line: max_parallel_downloads=10 and fastestmirror=True

Like That..

max_parallel_downloads=10
fastestmirror=True
